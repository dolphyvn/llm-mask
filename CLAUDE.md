# llm-mask

A data masking service for LLM inputs. Masks sensitive data (API keys, emails, PII) before sending to cloud LLMs while preserving semantic meaning.

## Project Goals

- **Primary**: Enable safe use of cloud LLMs with sensitive/internal data
- **Secondary**: Preserve data structure so LLM can still analyze patterns
- **Non-goal**: Encrypt/reversible masking (mappings are ephemeral, in-memory only)
